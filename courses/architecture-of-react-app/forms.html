# 表单系统

在 React 中，表单有受控open in new window和非受控open in new window两种。

  • React Hook Formopen in new window 基于非受控组件思路实现的表单系统，性能卓著
  • Formikopen in new window 基于受控组件思路实现的表单系统，功能强大
